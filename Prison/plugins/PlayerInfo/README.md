# PlayerInfo
Shows info about a certain player!

#### Command Usage:
`/playerinfo <player>`

We also have a shorter command:\
`/pinfo <player>`

#### Permission:
`playerinfo.use` - Allows the player to use the command `/playerinfo`

#### Example:

Name: Steve\
IP: 127.0.0.1\
Ping: 46ms\
OS: iOS\
Model: iPhone\
UI: Classic UI\
GUI Scale: Maximum\
Controls: Touch\
Health: 19HP\
Position: X: 132, Y: 78, Z: -963
